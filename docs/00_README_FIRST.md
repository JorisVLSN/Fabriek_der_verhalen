# Verhalenfabriek

Welkom in de wereld van Professor Pluis.

Verhalenfabriek is geen AI-project. Het is een warme wereld waar kinderen zich welkom voelen, waar fantasie groeit en waar verhalen ontstaan.

AI is slechts een hulpmiddel. De echte magie ontstaat door de ideeën van kinderen.

Iedere beslissing moet beantwoorden aan één vraag:

> Zou Professor Pluis hier trots op zijn?

## Kernzin

> Iedereen verdient een verhaal waarin hij of zij zich gezien voelt.

Als we ooit moeten kiezen tussen slimmer en warmer, kiezen we altijd voor warmer.
